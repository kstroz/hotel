# Hotel app

## Prerequisites

- NodeJS v21.7.3
- NPM v10.5.0

## Environment setup

- [Set Up Your Environment](https://reactnative.dev/docs/set-up-your-environment)
- Run `npm install`

## Daily workflow

- Run `npm run start`
- Run `npm run android` or `npm run ios`

## Testing

- Run `npm run test`
- To update snapshots run `npm run snap`

## Troubleshooting

- press `r` in terminal while metro server is running
- force kill application then restart it
- clear all caches by running `npm start --reset-cache`
